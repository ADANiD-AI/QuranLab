# Quran Lab - AI-Powered Quran Learning

## Overview
Quran Lab is a free, open-source AI app for Tajweed recitation analysis, memorization, and certification. Built with Next.js, Supabase, and Hugging Face models.

## Features
- Real-time Tajweed feedback (madd, ghunnah, idgham errors)
- 7 Qira'at support
- Teacher certification (85%+ score)
- Blockchain progress tracking

## Tech Stack
- Frontend: Next.js + Supabase Auth
- AI: Hugging Face (tarteel-ai/whisper-base-ar-quran, Habib-HF/tarbiyah-ai-v1-1)
- Workflow: Zapier Agent + GPT-4o-mini Chatboard

## Setup
1. `npm install`
2. `npm run dev`
